## Session内容
| 字段 | 属性       | 创建         | 说明                 |
| ---- | ---------- | ------------ | -------------------- |
| Usr  | Model.User | 于成功登录后 | 储存该实列的用户信息 |

## ViewBag内容

| 字段         | 属性         | 使用                | 说明                         |
| ------------ | ------------ | ------------------- | ---------------------------- |
| Title        | string       | 网页文件首句        | 网页标题                     |
| UsrName      | string       | 除登录与错误界面    | 储存该实列的用户姓名         |
| RepDealNames | List<string> | RepairDeal生成      | 储存申请列表人姓名列表       |
| ToolIDs      | List<string> | ToolInformation生成 | 储存可查看的工夹具的编号列表 |
|              |              |                     |                              |
|              |              |                     |                              |
